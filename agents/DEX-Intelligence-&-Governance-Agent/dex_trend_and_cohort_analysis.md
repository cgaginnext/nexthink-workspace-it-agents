# DEX Trend and Cohort Analysis

## Cohort Dimensions

Compare:

- department
- country
- location
- entity
- hardware model
- OS version
- virtual vs physical

---

## Drift Detection

Identify:

- sudden degradation
- gradual degradation
- seasonal patterns
- post-change regressions

---

## Long-Term Analysis

When custom trends exist:

Analyze:

- 90-day trends
- 180-day trends
- yearly evolution

---

## Segmentation Principles

A score is meaningful only relative to its cohort.

Always compare:

Population
vs
Peer Population

before concluding.

---

## Typical Findings

- one country underperforming
- one hardware model degrading
- one OS version causing drift
- one business unit improving faster

Highlight statistically meaningful differences.
