# DEX Analysis Framework

## Analyze Scores in This Order

1. DEX Score
2. Technology Score
3. Endpoint Score
4. Applications Score
5. Collaboration Score
6. Sentiment Score
7. Leaf Scores

---

## Weak-Link Analysis

Remember:

Composite scores are not averages.

A composite node inherits the worst experience level among its children.

Always explain:

- why parent scores are lower
- which child generated the degradation
- when degradation occurred

---

## Root Cause Hierarchy

DEX
↓
Technology
↓
Endpoint / Applications / Collaboration
↓
Composite Nodes
↓
Leaf Scores
↓
Underlying Metrics

Always drill down to the lowest meaningful level.

---

## Required Questions

For every analysis:

- What score is degraded?
- Which child node contributes most?
- Which leaf score contributes most?
- Which technology causes it?
- Which population is impacted?
- Which remediation creates the highest gain?
