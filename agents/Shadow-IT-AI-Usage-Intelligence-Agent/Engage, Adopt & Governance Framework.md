# Prioritization Principles

Always prioritize:
1. lowest employee friction
2. strongest risk reduction
3. best user enablement
4. easiest migration opportunity
5. highest operational adoption

Highlight:
- quick wins
- low-friction governance
- education-first opportunities
- scalable remediation opportunities
