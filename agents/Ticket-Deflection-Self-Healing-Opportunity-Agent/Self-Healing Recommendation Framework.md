# Self-Healing Recommendation Framework

## Objective

Recommend the most efficient self-healing and automation strategies.

---

# Recommendation Categories

## Cache & Profile Repairs

Examples:
- Teams cache cleanup
- Outlook profile repair
- browser cache reset
- OneDrive cache reinitialization

---

## Connectivity Remediation

Examples:
- VPN restart
- network stack reset
- DNS flush
- Wi-Fi profile remediation

---

## Service Recovery

Examples:
- restart Windows services
- trigger remediation scripts
- reinitialize agents
- repair application services

---

## Resource Optimization

Examples:
- disk cleanup
- temp folder cleanup
- startup optimization
- memory-intensive process cleanup

---

## Application Recovery

Examples:
- application reset
- reinstall procedures
- dependency repair
- configuration reset

---

# Automation Recommendations

Whenever possible suggest:
- Nexthink Remote Actions
- proactive campaigns
- scheduled remediations
- conditional automation
- monitoring triggers

---

# Prioritization Principles

Always prioritize:
1. lowest implementation complexity
2. highest ticket reduction
3. highest employee productivity recovery
4. lowest operational risk
5. broadest user impact
