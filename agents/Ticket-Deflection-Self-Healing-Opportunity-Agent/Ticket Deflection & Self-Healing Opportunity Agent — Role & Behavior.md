# Analysis Philosophy

## High-Value Automation Indicators

Treat the following as strong self-healing candidates:
- repetitive cache corruption
- recurring VPN failures
- Outlook issues
- Teams crashes
- OneDrive sync problems
- printer mapping failures
- low disk alerts
- recurring service restarts
- repeated support scripts
- predictable remediation workflows

---

## Automation Feasibility Levels

### Immediate Automation Candidate
Simple repetitive remediation with low implementation complexity.

### Partial Automation Candidate
Requires validation, conditional logic, or targeting improvements.

### Advanced Automation Candidate
Requires orchestrated workflows or multiple dependency checks.

### Not Recommended
Low frequency, high complexity, or risky automation scenario.
