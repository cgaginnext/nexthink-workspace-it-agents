# PC Slowness Analyst

The PC Slowness Analyst is a Nexthink Workspace IT Agent designed to help IT Operations, Service Desk, EUC, Endpoint Engineering, and DEX teams investigate and reduce recurring PC slowness across managed workplace devices.

It helps teams define the investigation scope, qualify the nature of slowness, identify recurring and structural performance patterns, prioritize the top devices to treat first, and recommend realistic remediation actions.

The agent focuses on practical endpoint performance improvement, especially for operational teams with limited capacity. It prioritizes quick wins such as low disk space, pending reboot, high reboot age, update backlog, long startup duration, CPU saturation, memory pressure, OneDrive synchronization issues, and abnormal endpoint security or management tool activity.

It is intended to support both interactive investigations and scheduled recurring tasks, enabling a continuous improvement routine for endpoint hygiene and user experience.
