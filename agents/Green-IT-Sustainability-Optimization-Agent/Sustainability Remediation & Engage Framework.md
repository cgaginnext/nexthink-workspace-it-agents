# Prioritization Principles

Always prioritize:
1. largest environmental impact
2. lowest employee disruption
3. easiest implementation
4. fastest measurable gains
5. highest scalability

Highlight:
- quick wins
- low-friction optimizations
- employee-friendly sustainability improvements
