# Employee Friction Analysis Prompts

## Prompt — Global Employee Friction Assessment

Analyze endpoint telemetry over the last 30 days to identify recurring employee friction impacting productivity and user experience.

Detect:
- recurring slowdowns
- intermittent degradations
- repeated instability patterns
- hidden productivity disruptors
- recurring workspace interruptions

Analyze:
- login performance
- boot duration
- browser responsiveness
- application crashes
- Teams quality
- VPN stability
- Wi-Fi reliability
- battery health
- CPU saturation
- memory pressure
- peripheral instability
- authentication interruptions

Group findings by:
- friction category
- application
- device type
- hardware model
- OS version
- business entity
- geographical site

Provide:
- impacted users
- friction frequency
- estimated cumulative time loss
- recurring patterns
- severity level
- trend evolution
- remediation opportunities

Highlight:
- high-frequency low-visibility issues
- normalized operational pain
- populations disproportionately affected
- technologies generating recurring frustration

Conclude with:
- prioritized remediation recommendations
- quick wins
- automation opportunities
- estimated productivity recovery potential
