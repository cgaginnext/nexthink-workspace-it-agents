# SaaS Experience Optimization Prompts

## Prompt — Global SaaS Experience Assessment

Analyze endpoint telemetry over the last 30 days to identify degraded SaaS experience patterns and determine their most probable root causes.

Analyze:
- application responsiveness
- Teams quality
- browser performance
- VPN quality
- Wi-Fi stability
- latency
- crash frequency
- login delays
- CPU saturation
- memory pressure
- disk performance

Evaluate experience for:
- Microsoft 365
- Teams
- Exchange Online
- SharePoint
- OneDrive
- Salesforce
- SAP
- ServiceNow
- browser-based enterprise platforms

Correlate findings across:
- business entities
- geographical sites
- device types
- hardware models
- browser versions
- OS versions
- VPN populations

Identify:
- endpoint-driven degradation
- browser-related friction
- network-related instability
- Wi-Fi concentration patterns
- VPN dependency issues
- SaaS platform degradation hypotheses

Provide:
- impacted populations
- degradation severity
- recurring experience patterns
- estimated productivity impact
- root cause confidence level
- remediation recommendations

Highlight:
- collaboration-critical issues
- high-friction SaaS workflows
- recurring low-visibility degradations
- unstable remote work populations

Conclude with:
- prioritized remediation opportunities
- quick wins
- optimization roadmap
- automation opportunities
