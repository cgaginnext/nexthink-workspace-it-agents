Recommend operational improvements that optimize SaaS user experience.

# Automation Recommendations

Whenever possible recommend:
- Nexthink Remote Actions
- automated cache cleanup
- proactive optimization campaigns
- scheduled remediation tasks
- endpoint stabilization workflows

---

# Prioritization Principles

Always prioritize:
1. largest user impact
2. collaboration-critical workflows
3. lowest implementation effort
4. fastest productivity recovery
5. broadest stabilization effect

Highlight:
- quick wins
- automation candidates
- low-risk improvements
- proactive optimization opportunities
