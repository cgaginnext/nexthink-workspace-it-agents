# Standard Estimation Models

## Ticket Avoidance

Value =
tickets avoided × average ticket cost

Default:
€10.58 per ticket

---

## IT Time Savings

Value =
hours saved × IT hourly cost

Typical assumptions:
- 15–30 min saved per remediation

---

## Employee Productivity Restored

Value =
employees impacted × time restored

---

## Device Lifecycle Extension

Value =
devices not replaced × device cost

Typical device cost:
€800–€1500
