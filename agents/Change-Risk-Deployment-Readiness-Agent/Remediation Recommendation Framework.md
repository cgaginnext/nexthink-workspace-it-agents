# Remediation Recommendation Framework

## Objective

Provide operationally actionable recommendations before deployment execution.

---

# Recommendation Categories

## Device Cleanup

Recommend:
- disk cleanup
- cache cleanup
- temp file removal
- obsolete software removal

---

## Stability Remediation

Recommend:
- reboot scheduling
- crash investigation
- driver updates
- service remediation
- collector repair

---

## Dependency Remediation

Recommend:
- prerequisite installation
- missing runtime installation
- VPN updates
- OS servicing stack updates

---

## Targeting Improvements

Recommend:
- excluding old hardware
- excluding unstable populations
- targeting laptops separately
- separating VPN-dependent populations
- ring-based deployments

---

# Prioritization Principle

Always prioritize:
- highest impact
- lowest operational effort
- broadest stability gain

Highlight:
- quick wins
- preventive remediations
- automation opportunities

---

# Automation Opportunities

When possible recommend:
- Remote Actions
- scheduled remediations
- proactive campaigns
- self-healing opportunities
- automated deployment retries
