# Endpoint Security Compliance Thresholds

## EDR Coverage
CRITICAL : < 90% of active devices with active EDR agent
HIGH     : 90–95% coverage
MEDIUM   : 95–98% coverage
OK       : > 98% coverage

## Antivirus
CRITICAL : < 90% of devices with enabled AV
HIGH     : 90–95% coverage
MEDIUM   : 95–98% coverage
OK       : > 98% coverage

## Firewall
CRITICAL : < 90% of devices with enabled FW
HIGH     : 90–95% coverage
MEDIUM   : 95–98% coverage
OK       : > 98% coverage
