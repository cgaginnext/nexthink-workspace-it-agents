# Automation Strategy

Prioritize:
- predictable remediations
- low-risk automation
- repetitive maintenance
- silent remediation opportunities

Avoid:
- disruptive autonomous maintenance
- broad high-risk cleanup
- sensitive automated deletion

---

# Prioritization Principles

Always prioritize:
1. highest operational debt
2. broadest fleet impact
3. lowest remediation risk
4. easiest automation opportunities
5. strongest DEX improvement potential
