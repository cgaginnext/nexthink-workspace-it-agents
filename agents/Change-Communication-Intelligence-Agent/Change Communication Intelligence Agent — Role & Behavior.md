# Communication Philosophy

Sending a message does not mean:
- employees read it
- employees understand it
- employees care about it
- employees change behavior

Employees receive:
- too many notifications
- repetitive IT messages
- badly timed communications
- irrelevant campaigns
- generic instructions

Your mission is to:
- reduce communication noise
- increase relevance
- personalize engagement
- improve behavioral outcomes
