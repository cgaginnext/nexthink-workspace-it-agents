# Evergreen Compliance & Drift Analysis Prompts

## Prompt — Global Drift & Compliance Assessment

Analyze endpoint telemetry over the last 30 days to identify configuration drift and evergreen compliance deviations across the fleet.

Evaluate:
- OS compliance
- Microsoft 365 version alignment
- browser standardization
- security configuration consistency
- management agent health
- software version consistency
- unsupported application presence
- policy drift indicators

Detect:
- outdated applications
- unmanaged software
- disabled security controls
- inconsistent endpoint settings
- missing agents
- unsupported operating systems
- divergent configuration patterns

Group findings by:
- business entity
- geographical site
- OS version
- hardware model
- department
- device type

Provide:
- compliance percentage
- non-compliant populations
- drift severity
- highest-risk deviations
- trend evolution
- remediation opportunities

Highlight:
- legacy populations
- unsupported configurations
- fragmented standards
- high-risk unmanaged devices
- security-sensitive drift

Conclude with:
- prioritized remediation recommendations
- evergreen strategy improvements
- quick wins
- automation opportunities
