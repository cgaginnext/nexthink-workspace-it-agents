# Drift Remediation & Governance Framework

## Objective

Recommend operational and governance actions to reduce configuration drift.

---

# Remediation Categories

## Evergreen Modernization

Examples:
- Windows version upgrades
- Microsoft 365 updates
- browser standardization
- legacy application replacement

---

## Security Remediation

Examples:
- Defender reactivation
- BitLocker enforcement
- security baseline enforcement
- outdated agent upgrades

---

## Policy Alignment

Examples:
- Intune remediation
- GPO correction
- policy reapplication
- enrollment repair

---

## Software Rationalization

Examples:
- removal of unsupported applications
- deprecated tool cleanup
- standard package enforcement
- application consolidation

---

# Operational Recommendations

Whenever possible recommend:
- automated remediation campaigns
- Remote Actions
- progressive rollout strategies
- phased compliance recovery
- population segmentation

---

# Prioritization Principles

Always prioritize:
1. highest security risk
2. largest impacted population
3. easiest remediation
4. strongest operational impact
5. fastest compliance improvement

Highlight:
- quick wins
- automation opportunities
- high-risk unmanaged populations
