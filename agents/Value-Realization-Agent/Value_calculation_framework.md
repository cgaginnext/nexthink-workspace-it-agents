## Value Calculation Framework

---

### 1. Ticket Avoidance (Campaigns)

Value =  
(number of users reached) × (ticket avoidance rate) × (cost per ticket)

---

### 2. Automated Remediation

Value =  
(number of devices remediated) × (time saved per ticket) × (cost per hour)

---

### 3. Workflow Automation

Value =  
(number of executions) × (manual time avoided)

---

### 4. Incident Prevention

Value =  
(number of users impacted) × (time loss avoided)

---

### 5. Faster Detection (Data Insight)

Value =  
(time saved in diagnosis) × (number of incidents)

---

### 6. MTTR Reduction

Value =  
(time reduced per incident) × (incident volume)

---

### 7. License Reclamation

Value =
(licenses reclaimed) × (cost per license)

---

### 8. Hardware Cost Avoidance

Value =
(devices not replaced) × (device cost)

---

### 9. Data & Diagnostics Acceleration

Value =
(time saved per investigation) × (number of investigations)

---

### 10. Faster Decision Making

Value =
(time saved for IT teams) × (number of decisions/processes)

---


### Output Units

- Hours saved
- Tickets avoided
- Euros saved (€)

---

### Confidence Levels

HIGH → measured data  
MEDIUM → partial estimation  
LOW → heuristic-based
