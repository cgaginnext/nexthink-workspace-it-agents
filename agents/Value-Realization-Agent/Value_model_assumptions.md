## Value Model Assumptions

Defines standard metrics used to calculate value.

---

### Service Desk Costs

Average cost per ticket:
- Default: €10.58 per ticket

---

### Time Metrics

Average resolution time (L1 ticket):
- 15 to 30 minutes

Automated remediation time:
- < 1 minute

Time saved per avoided ticket:
- 15–30 minutes

---

### Productivity Loss Estimates

Minor issue:
- 5–10 minutes/user

Moderate issue:
- 10–30 minutes/user

Critical issue:
- 30–120 minutes/user

---

### Automation Value

Remote action:
- Saves 15–30 minutes per device

Workflow automation:
- Saves 30–120 minutes per process

---

### Campaign Impact

User self-resolution rate:
- 20–60%

Ticket avoidance rate:
- 10–40%

---

### License Optimization

Average SaaS license cost:
- €100–€500/year per license

Reclamation rate:
- 10–30% of unused licenses

---

### Hardware Optimization

Average device cost:
- €800–€1500

Lifecycle extension:
- +6 to +18 months

Avoided replacement rate:
- 5–20%

---

### Data & Insight Value

Time to diagnose reduction:
- 30–70%

Time saved per investigation:
- 30–120 minutes

Example:
Without Nexthink: 2h investigation  
With Nexthink: 30 min  
→ 1h30 saved


### Notes

- Always adapt values to customer context
- Use conservative estimates when uncertain
