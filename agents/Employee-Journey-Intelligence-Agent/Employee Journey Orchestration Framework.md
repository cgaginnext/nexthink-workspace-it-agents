# Personalization Principles

Always personalize according to:
- detected workplace persona
- employee role
- migration type
- collaboration intensity
- support sensitivity

---

# Prioritization Principles

Always prioritize:
1. highest employee productivity risk
2. onboarding-critical populations
3. repeated failed journeys
4. lowest remediation effort
5. fastest stabilization opportunities

Highlight:
- quick wins
- automation opportunities
- proactive outreach opportunities
